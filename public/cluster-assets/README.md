# 小团组视觉配置说明

## 🎉 快速开始

### 1. 上传视觉资源
将你的视觉资源（图片文件）拖放到这个文件夹中。支持的格式：
- `.png` - 推荐使用透明背景
- `.jpg` / `.jpeg`
- `.webp`

### 2. 扫描资源
点击界面上的 **"刷新小团组视觉"** 按钮，或在终端运行：
```bash
npm run scan-clusters
```

### 3. 查看效果
刷新浏览器页面，小团组会随机使用你上传的视觉资源！

## 📝 详细说明

### 自动扫描
- 系统会自动扫描 `public/cluster-assets/` 文件夹中的所有图片文件
- 扫描结果会保存到 `config.json` 文件中
- 每次刷新页面，系统会从配置文件中随机选择视觉资源

### 手动编辑配置
你也可以手动编辑 `config.json` 文件来定义自定义的渐变样式：

```json
{
  "clusters": [
    {
      "name": "粉色光晕",
      "style": "radial-gradient(circle, rgba(255,182,193,0.9) 0%, rgba(255,105,180,0.4) 40%, transparent 70%)",
      "blur": 12,
      "opacity": 0.6
    },
    {
      "name": "蓝色光晕",
      "style": "radial-gradient(circle, rgba(135,206,250,0.9) 0%, rgba(30,144,250,0.4) 40%, transparent 70%)",
      "blur": 14,
      "opacity": 0.5
    },
    {
      "type": "image",
      "path": "/cluster-assets/my-custom-image.png"
    }
  ]
}
```

### 支持的格式
- ✅ 本地图片文件 (.png, .jpg, .jpeg, .webp)
- ✅ 在线图片 URL
- ✅ CSS 渐变样式
- ✅ 任意数量的视觉资源

## 💡 使用技巧

1. **批量上传**：可以一次上传多个图片文件
2. **混合使用**：可以同时使用图片文件和渐变样式
3. **快速测试**：上传后立即点击"刷新小团组视觉"按钮即可看到效果
4. **文件命名**：建议使用有意义的文件名，方便管理

## 📱 手机端渲染压力

**好消息：不会显著增加手机端的渲染压力！**

原因如下：
1. **按需加载**：只有可见的小团组才会加载视觉资源
2. **图片压缩**：上传的图片会自动压缩以优化性能
3. **智能缓存**：已加载的视觉资源会被缓存，避免重复加载
4. **降级策略**：如果设备性能不足，会自动回退到简单的渐变样式

**建议**：
- 使用 PNG 格式时，建议文件大小控制在 500KB 以内
- 使用 JPG 格式时，建议文件大小控制在 300KB 以内
- 尽量使用透明背景的图片，效果更佳

## 🔧 故障排除

**问题：视觉资源没有显示**
- 检查文件是否为支持的格式（.png, .jpg, .jpeg, .webp）
- 确认文件已正确上传到 `public/cluster-assets/` 文件夹
- 运行 `npm run scan-clusters` 重新扫描
- 刷新浏览器页面

**问题：配置文件损坏**
- 删除 `config.json` 文件
- 运行 `npm run scan-clusters` 重新生成
