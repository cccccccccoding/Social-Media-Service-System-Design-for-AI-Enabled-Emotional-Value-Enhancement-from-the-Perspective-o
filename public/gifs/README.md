# 动图配置说明

## 🎉 快速开始

### 1. 上传动图
将你的 `.gif` 动图文件拖放到这个文件夹中

### 2. 扫描动图
点击界面右上角的 **"刷新动图"** 按钮，或在终端运行：
```bash
npm run scan-gifs
```

### 3. 查看效果
刷新浏览器页面，动图会随机显示！

## 📝 详细说明

### 自动扫描
- 系统会自动扫描 `public/gifs/` 文件夹中的所有 `.gif` 文件
- 扫描结果会保存到 `config.json` 文件中
- 每次刷新页面，系统会从配置文件中随机选择动图

### 手动编辑配置
你也可以手动编辑 `config.json` 文件：

```json
{
  "gifs": [
    "/gifs/my-gif-1.gif",
    "/gifs/my-gif-2.gif",
    "https://example.com/online-gif.gif"
  ]
}
```

### 支持的格式
- ✅ 本地 `.gif` 文件
- ✅ 在线 GIF URL
- ✅ 任意数量的动图

## 💡 使用技巧

1. **批量上传**：可以一次上传多个动图文件
2. **混合使用**：可以同时使用本地文件和在线URL
3. **快速测试**：上传后立即点击"刷新动图"按钮即可看到效果
4. **文件命名**：建议使用有意义的文件名，方便管理

## 🔧 故障排除

**问题：动图没有显示**
- 检查文件是否为 `.gif` 格式
- 确认文件已正确上传到 `public/gifs/` 文件夹
- 运行 `npm run scan-gifs` 重新扫描
- 刷新浏览器页面

**问题：配置文件损坏**
- 删除 `config.json` 文件
- 运行 `npm run scan-gifs` 重新生成

## 📱 手机访问

手机也可以访问和查看动图效果，确保：
- 手机和电脑连接到同一个Wi-Fi
- 使用正确的局域网地址访问
